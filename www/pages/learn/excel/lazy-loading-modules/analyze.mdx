## Analyze

The app will look like this and it is a pretty basic one:

![](/static/images/llm-hackernews.png)

Now, let's try to understand what is inside our app bundle.

For that, simply run the following command:

```bash
npm run analyze:browser
```

Then, it will start a [webpack bundle analyzer](https://github.com/zeit/next-plugins/tree/master/packages/next-bundle-analyzer) and you will be able to inspect what is inside each of the JavaScript bundles.
